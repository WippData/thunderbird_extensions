# thunderbird_extensions
These are thunderbird desktop client extensions that we've built that we love. These are provided fully open source for anyone to use, modify, and distribute as they please.

If you want to contribute, fork the repo and submit a PR. 


Instructions: 
1. Each Folder is a distinct and dedicated plugin. 
2. Download and Zip any individual folder.
3. Install onto your wordpress instance.
